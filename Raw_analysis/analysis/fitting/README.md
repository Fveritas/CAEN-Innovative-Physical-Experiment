# 拟合分析 (Fitting Analysis)

本目录包含数据拟合和拟合质量评估的脚本。

## 脚本说明

### `fit_analysis.py`
**功能**: 对实验数据进行各种函数拟合

**支持的拟合模型**:

1. **线性拟合**
   ```
   y = a + bx
   ```
   - 用于趋势分析
   - 检验线性关系

2. **指数衰减**
   ```
   y = A × exp(-λx)
   ```
   - 衰减过程
   - 半衰期测量

3. **高斯分布**
   ```
   y = A × exp(-(x-μ)²/(2σ²))
   ```
   - 峰位分析
   - 分辨率测量

4. **多项式拟合**
   ```
   y = a₀ + a₁x + a₂x² + ...
   ```
   - 复杂曲线拟合
   - 背景建模

**拟合方法**:
- 最小二乘法 (Least Squares)
- 加权拟合（考虑误差）
- 鲁棒拟合（抗异常值）

**输出**:
- 拟合参数及误差
- 协方差矩阵
- 拟合质量指标（χ², R²）
- 残差分析

**用法**:
```bash
python fit_analysis.py
```

---

### `analyze_fit_quality.py`
**功能**: 评估拟合质量和可靠性

**评估指标**:

1. **χ² 检验**
   ```
   χ² = Σ [(y_i - f(x_i))² / σ_i²]
   ```
   - χ²/ndf ≈ 1: 好的拟合
   - χ²/ndf >> 1: 拟合不佳或误差低估
   - χ²/ndf << 1: 误差高估

2. **决定系数 R²**
   ```
   R² = 1 - SS_res/SS_tot
   ```
   - R² → 1: 拟合很好
   - R² → 0: 拟合很差

3. **残差分析**
   - 残差分布应随机
   - 无系统性偏离
   - 正态分布检验

4. **参数不确定度**
   - 参数误差
   - 相关系数
   - 置信区间

**诊断图表**:
- 数据 vs 拟合曲线
- 残差图（residual plot）
- 残差直方图
- Q-Q图（正态性检验）

**输出**:
- `fit_diagnostics.png` - 拟合诊断图（4面板）

**用法**:
```bash
python analyze_fit_quality.py
```

## 拟合诊断图解读

### 面板1: 数据与拟合曲线
- **蓝点**: 实验数据
- **红线**: 拟合曲线
- **误差棒**: 数据不确定度
- **检查**: 拟合曲线是否通过数据点

### 面板2: 残差图
- **横轴**: 自变量x
- **纵轴**: 残差 (y_data - y_fit)
- **理想**: 残差随机分布在0附近
- **警告**: 系统性趋势表明模型不合适

### 面板3: 残差直方图
- **形状**: 应接近正态分布
- **中心**: 应在0附近
- **宽度**: 反映拟合精度
- **异常**: 长尾或多峰表明问题

### 面板4: Q-Q图
- **理想**: 点落在对角线上
- **偏离**: 表明残差非正态分布
- **应用**: 检验统计假设

## 拟合策略

### 选择合适的模型

1. **物理动机**
   - 优先使用有物理意义的模型
   - 避免过度拟合

2. **模型复杂度**
   - 从简单模型开始
   - 逐步增加复杂度
   - 使用AIC/BIC准则

3. **参数数量**
   - 参数数 << 数据点数
   - 避免过参数化

### 初值选择

- **线性**: 用数据端点估计斜率
- **指数**: 用对数变换估计
- **高斯**: 用峰位和宽度估计
- **技巧**: 先粗略拟合，再精细调整

### 权重设置

1. **无权重**: 所有点等权
2. **泊松权重**: w_i = 1/y_i
3. **测量误差**: w_i = 1/σ_i²
4. **自定义**: 根据数据质量

## 常见问题诊断
### 拟合不收敛
**原因**:
- 初值选择不当
- 模型与数据不匹配
- 数据质量差

**解决**:
- 改进初值估计
- 尝试不同模型
- 检查数据异常值

### χ²/ndf >> 1
**原因**:
- 模型不合适
- 误差低估
- 存在系统误差

**解决**:
- 改用更复杂模型
- 重新评估误差
- 检查系统效应

### χ²/ndf << 1
**原因**:
- 误差高估
- 数据点相关性

**解决**:
- 检查误差计算
- 考虑相关性

### 残差有趋势
**原因**:
- 模型缺少某项
- 背景未正确扣除

**解决**:
- 增加多项式项
- 改进背景模型

## 应用示例

### 1. Rossi曲线拟合

**模型**: 
```python
def rossi_model(x, N0, x_max, lambda_abs):
    """
    x: 厚度
    N0: 归一化常数
    x_max: 峰位
    lambda_abs: 吸收长度
    """
    return N0 * (x/x_max) * np.exp((x_max - x)/lambda_abs)
```

**初值**:
- N0: 最大计数率
- x_max: 2-3 X₀
- lambda_abs: 1 X₀

### 2. 衰减曲线拟合

**模型**:
```python
def attenuation(x, N0, mu):
    """
    x: 厚度
    N0: 初始计数率
    mu: 衰减系数
    """
    return N0 * np.exp(-mu * x)
```
**初值**:
- N0: x=0处的计数率
- mu: -ln(N_end/N0)/x_max

### 3. 速度分布拟合

**模型**:
```python
def velocity_dist(v, v0, sigma):
    """
    v: 速度
    v0: 峰位速度
    sigma: 分布宽度
    """
    return np.exp(-(v - v0)**2 / (2*sigma**2))
```

**初值**:
- v0: 0.99c
- sigma: 从数据宽度估计

## 高级技巧

### 1. 联合拟合
同时拟合多个数据集，共享部分参数：
```python
# 例如：不同阈值数据共享衰减系数
def joint_fit(x1, x2, N01, N02, mu_shared):
    y1 = N01 * np.exp(-mu_shared * x1)
    y2 = N02 * np.exp(-mu_shared * x2)
    return y1, y2
```

### 2. 约束拟合
对参数施加物理约束：
```python
# 例如：衰减系数必须为正
bounds = ([0, 0], [np.inf, np.inf])  # (N0>0, mu>0)
```

### 3. 贝叶斯拟合
使用先验信息：
```python
# 例如：已知衰减系数的大致范围
prior = lambda mu: np.exp(-(mu - 0.02)**2 / (2*0.01**2))
```

### 4. Bootstrap方法
评估参数不确定度：
```python
# 重采样数据，多次拟合
params_bootstrap = []
for i in range(1000):
    data_resampled = resample(data)
    params_bootstrap.append(fit(data_resampled))
```

## 统计检验

### F检验
比较两个模型：
```
F = [(χ²₁ - χ²₂)/(df₁ - df₂)] / [χ²₂/df₂]
```
- 判断增加参数是否显著改善拟合

### t检验
检验参数是否显著非零：
```
t = (param - 0) / σ_param
```
- |t| > 2: 参数显著

### Kolmogorov-Smirnov检验
检验残差分布：
```python
from scipy.stats import kstest
ks_stat, p_value = kstest(residuals, 'norm')
```
- p > 0.05: 残差符合正态分布

## 常见问题

**Q: 如何判断拟合是否可靠？**
A: 综合考虑χ²/ndf、残差分布、参数误差和物理合理性。

**Q: 参数误差很大怎么办？**
A: 增加数据量、改进模型或检查数据质量。

**Q: 如何选择拟合范围？**
A: 排除明显异常点，保留物理相关区域。

**Q: 多个模型都拟合得很好，如何选择？**
A: 优先选择物理动机强、参数少的模型。
