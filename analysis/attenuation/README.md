# 衰减分析 (Attenuation Analysis)

本目录包含宇宙射线在铅中衰减特性的分析脚本。

## 物理背景

### 粒子在物质中的衰减
带电粒子穿过物质时通过以下机制损失能量：
1. **电离损失** (主要): dE/dx ∝ ρZ/A
2. **韧致辐射**: 高能电子的主要损失机制
3. **多重散射**: 改变粒子方向
4. **核相互作用**: 强子的额外损失

对于宇宙射线μ子：
- 主要通过电离损失能量
- 在铅中: dE/dx ≈ 1.5 MeV/(g/cm²)
- 射程: 几十到几百g/cm²（取决于能量）

## 脚本说明

### `analyze_lead_attenuation.py`
**功能**: 分析宇宙射线符合率随铅厚度的衰减规律

**分析内容**:
1. **指数衰减拟合**: N(x) = N₀ × exp(-μx)
2. **线性衰减系数**: μ (单位: cm⁻¹ 或 g⁻¹cm²)
3. **半值层厚度**: x₁/₂ = ln(2)/μ
4. **平均自由程**: λ = 1/μ

**输入数据**:
- 不同铅板厚度下的符合计数率
- 测量时间和统计误差

**输出**:
- 衰减曲线图
- 拟合参数（N₀, μ）
- 半值层厚度
- χ²/ndf 拟合质量

**用法**:
```bash
python analyze_lead_attenuation.py
```

**物理意义**:
- **μ值**: 反映粒子与物质相互作用强度
- **半值层**: 使计数率减半所需的物质厚度
- **指数衰减**: 表明粒子能量分布和多重散射效应

---

### `plot_attenuation.py`
**功能**: 绘制衰减曲线的可视化工具

**图表类型**:
1. **线性坐标**: 直观显示衰减趋势
2. **半对数坐标**: 检验指数衰减（应为直线）
3. **归一化曲线**: 相对于0厚度归一化

**特性**:
- 误差棒显示（泊松统计）
- 拟合曲线叠加
- 残差分析
- 多数据集对比

**输出**:
- `lead_attenuation.png` - 衰减曲线图

**用法**:
```bash
python plot_attenuation.py
```

## 理论模型

### 简单指数衰减
```
N(x) = N₀ × exp(-μx)
```
- N(x): 厚度x处的计数率
- N₀: 初始计数率（x=0）
- μ: 线性衰减系数
- x: 物质厚度

### 考虑能量分布
实际宇宙射线有能量谱，不同能量粒子衰减系数不同：
```
N(x) = ∫ N₀(E) × exp(-μ(E)x) dE
```
导致偏离简单指数衰减。

### 几何效应
- 多重散射使粒子偏离原方向
- 大角度散射的粒子可能miss下探测器
- 等效于额外的"衰减"

## 数据分析

### 预期结果
对于宇宙射线μ子在铅中：
- **衰减系数**: μ ≈ 0.01-0.03 cm⁻¹
- **半值层**: x₁/₂ ≈ 20-70 cm
- **平均自由程**: λ ≈ 30-100 cm

**注意**: 这些值远大于电磁簇射的特征长度（X₀ = 0.56 cm），说明μ子穿透能力强。

### 实验结果
根据0-30片铅板数据（0-15 mm）：
- 符合率下降: ~11% per 15mm
- 等效衰减系数: μ ≈ 0.08 cm⁻¹
- 半值层: x₁/₂ ≈ 8.7 cm

**解释**: 
- 衰减主要来自**几何效应**（多重散射）
- 而非能量损失（μ子射程远大于15mm）
- 探测器间距16.2 cm，散射角>10°的粒子会miss

---

## 系统效应

### 多重散射
散射角的RMS：
```
θ₀ = (13.6 MeV)/(βcp) × √(x/X₀) × [1 + 0.038×ln(x/X₀)]
```
对于1 GeV μ子穿过15mm铅：
- θ₀ ≈ 3.6°
- 在16.2 cm距离上横向偏移 ~1 cm
- 部分粒子散射出探测器

### 能量损失
1 GeV μ子穿过15mm铅：
- ΔE ≈ 1.5 MeV/cm × 1.5 cm ≈ 2.3 MeV
- 相对损失: 0.23%
- 对符合率影响很小

### 核相互作用
μ子核相互作用截面很小：
- 相互作用长度: λ_int ≈ 100 cm
- 在15mm铅中概率: ~0.15%
- 可忽略

**结论**: 观察到的衰减主要由多重散射的几何效应导致。

## 改进建议

### 提高测量精度
1. **增加统计量**: 延长测量时间
2. **更多数据点**: 增加铅板厚度采样点
3. **重复测量**: 评估系统误差

### 扩展厚度范围
- 当前: 0-25 mm (0-4.5 X₀)
- 建议: 0-100 mm (0-18 X₀)
- 可观察到更完整的衰减曲线

### 能量依赖研究
- 使用不同阈值选择不同能量范围
- 研究衰减系数与能量的关系
- 提取能量谱信息

### 角度分布
- 测量不同角度的符合率
- 研究多重散射角分布
- 验证理论预期

## 常见问题

**Q: 为什么衰减这么快？**
A: 主要是多重散射导致的几何损失，不是能量损失。

**Q: 如何区分能量损失和散射效应？**
A: 改变探测器间距，散射效应会显著变化，能量损失不变。

**Q: 能否用衰减曲线测量粒子能量？**
A: 原则上可以，但需要精确的散射模型和更多数据点。

**Q: 为什么不是完美的指数衰减？**
A: 宇宙射线有能量谱，不同能量粒子衰减系数不同。
