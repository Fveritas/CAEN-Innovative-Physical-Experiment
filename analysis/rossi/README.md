# Rossi转换曲线分析 (Rossi Transition Curve Analysis)

本目录包含Rossi转换曲线相关的分析脚本，用于研究宇宙射线在铅中的电磁簇射过程。

## 物理背景

### Rossi转换曲线
Rossi转换曲线描述了高能粒子穿过物质时，符合计数率随物质厚度的变化关系：

1. **初始阶段** (0 - 1 X₀): 电磁簇射发展，次级粒子数增加，符合率上升
2. **Rossi峰** (~2-3 X₀): 簇射达到最大值，符合率达到峰值
3. **衰减阶段** (>3 X₀): 粒子被吸收，符合率指数下降

**关键参数**:
- 辐射长度 X₀(Pb) = 5.6 mm
- 临界能量 Ec(Pb) ≈ 7.4 MeV
- 铅密度 ρ = 11.34 g/cm³

### `rossi_analysis_v2.py` ⭐ **推荐使用**
**功能**: 改进版Rossi曲线分析，考虑系统偏移

**改进点**:
1. 识别不同周次测量的系统差异
2. 分析下探测器效率变化
3. 归一化符合率（Coin/Up ratio）
4. 多阈值比较（threshold=5 vs 30）

**物理参数**:
```python
MEASUREMENT_TIME_HOURS = 1.0        # 每个数据点测量时间
PLATE_THICKNESS_MM = 0.5            # 单片铅板厚度
DETECTOR_DISTANCE_CM = 16.2         # 探测器间距
LEAD_RADIATION_LENGTH_MM = 5.6      # 铅的辐射长度
LEAD_DENSITY_G_CM3 = 11.34          # 铅密度
```

**输出图表** (4个子图):
1. **符合率 vs 厚度**: 显示不同周次数据，标注系统偏移
2. **下探测器触发率**: 诊断系统效率变化
3. **归一化符合率** (Coin/Up): 消除上探测器漂移影响
4. **阈值比较**: 对比不同阈值下的符合率

**关键发现**:
- 0-30片铅板数据（上周）内部一致
- 40-50片铅板数据（本周）存在系统偏移
- 下探测器效率下降约X%
- **未观察到明显Rossi峰**，符合率单调下降

**用法**:
```bash
python rossi_analysis_v2.py
```

### `why_no_rossi_curve.py`
**功能**: 诊断为什么没有观察到Rossi峰

**分析内容**:
1. **能量阈值效应**: 
   - 阈值太高可能过滤掉低能次级粒子
   - 只保留高能穿透粒子

2. **探测器几何**:
   - 探测器间距16.2 cm
   - 立体角覆盖范围
   - 次级粒子散射角度

3. **入射粒子能量**:
   - 宇宙射线μ子能量分布
   - 是否足够产生显著簇射

4. **物质厚度范围**:
   - 0-30片 = 0-15 mm = 0-2.7 X₀
   - 可能未达到簇射最大值

**可能原因**:
- ✓ 入射粒子能量不足（μ子主要通过电离损失能量）
- ✓ 探测器间距太小，次级粒子散射出探测器
- ✓ 阈值设置过滤了低能次级粒子
- ✓ 铅板厚度范围不够（需要更厚的铅）

**改进建议**:
1. 增加探测器间距（如50 cm）
2. 降低触发阈值
3. 增加铅板厚度（至少到5 X₀）
4. 使用更大面积探测器

## 理论预期 vs 实验结果

### 理论预期
- Rossi峰位置: ~2-3 X₀ (11-17 mm铅)
- 峰值增强: 20-50%（取决于入射能量）
- 衰减长度: ~1 X₀

### 实验结果
- **未观察到Rossi峰**
- 符合率单调下降
- 下降率: ~11% per 15mm铅
- 归一化后仍无峰值结构

### 物理解释
宇宙射线μ子主要通过**电离损失能量**，而非电磁簇射：
- μ子质量 = 105.7 MeV/c² (电子的207倍)
- 临界能量远高于典型宇宙射线μ子能量
- 韧致辐射和正负电子对产生截面很小
- 因此不产生显著的电磁簇射

**结论**: 要观察Rossi曲线，需要使用高能电子或γ射线源，而非宇宙射线μ子。

## 进阶分析

### 归一化方法
1. **Coin/Up ratio**: 消除上探测器计数率漂移
2. **Coin/Entries**: 相对于总触发数归一化
3. **时间归一化**: 考虑测量时间差异

### 系统误差
- 探测器效率漂移: ~10-15%
- 温度效应: PMT增益变化
- 高压稳定性: 影响阈值
- 几何位置: 铅板放置精度

### 统计误差
- 泊松统计: σ = √N
- 1小时测量: ~3000 counts → σ ≈ 55 (1.8%)
- 相对误差小，系统误差占主导

## 常见问题

**Q: 为什么没有看到Rossi峰？**
A: 宇宙射线μ子不产生显著电磁簇射，需要使用电子或γ源。

**Q: 如何改进实验？**
A: 使用放射源（如⁶⁰Co）、增加探测器间距、降低阈值。

**Q: 数据中的系统偏移是什么原因？**
A: 可能是PMT高压变化、温度效应或探测器位置调整。

**Q: 归一化后为什么还是没有峰？**
A: 说明不是系统漂移问题，而是物理过程本身不产生簇射。
